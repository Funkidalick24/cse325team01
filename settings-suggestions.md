# Settings Suggestions

## Profile
- Display name
- Email (read-only or editable with verification)
- Time zone
- Preferred date format

## Dashboard Preferences
- Default dashboard view (`Inbox`, `Today`, `Upcoming`, `Filters`, `Archive`)
- Default sort order (due date, priority, newest)
- Show/hide completed tasks on dashboard
- Compact mode toggle for task cards

## Notifications
- Daily summary reminder time
- Due-soon reminders (e.g., 1 day before)
- Overdue alerts
- In-app vs email notification toggles

## Task Defaults
- Default priority for new tasks
- Default category for new tasks
- Auto-set due date for quick-add tasks

## Security
- Change password
- Active sessions list + revoke session
- Optional 2FA/MFA toggle

## Data Management
- Export tasks (CSV/JSON)
- Import tasks
- Archive-completed cleanup rule (e.g., after 30/60/90 days)
- Account/data deletion request

## Appearance
- Theme selection (light/dark/system)
- Accent color preset
- Font size scaling

## Admin/Advanced (Optional)
- Maintenance mode toggle
- Audit log viewer
- Environment diagnostics (non-secret)
