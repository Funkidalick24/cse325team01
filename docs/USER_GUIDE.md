# Taskflow User Guide

## Getting Started
1. Open Taskflow in your browser.
2. Select `Register` and create an account.
3. Log in, then open `Dashboard`.

## Core Workflow
1. Select `Create New Task`.
2. Fill in title, optional description, due date, priority, and category.
3. Save the task.
4. Use the dashboard to:
   - Mark tasks complete.
   - Edit tasks.
   - Filter by status, priority, category, and overdue state.

## Settings Features
- `Profile`: update username, email, and phone.
- `Security`: change password.
- `Appearance`: switch between system/light/dark themes.
- `Data Management`:
  - Export tasks to JSON.
  - Delete completed tasks.
  - Delete all tasks by entering `DELETE ALL`.

## Error and Recovery Experience
- Form validation feedback appears before submission for invalid input.
- Submission failures show clear alert messages with retry guidance.
- Connection interruptions show a reconnect modal with retry/resume options.
- Unexpected server errors route to a dedicated error page.

## Accessibility Notes
- Keyboard users can use `Skip to main content` at the top of each page.
- Interactive controls have visible focus styles.
- Alerts include accessible live-region behavior for status/error messages.
